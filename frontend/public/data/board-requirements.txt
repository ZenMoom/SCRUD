기본 게시판 요구사항 명세서

1. 개요
사용자들이 글을 작성하고, 조회하고, 수정하고, 삭제할 수 있는 간단한 게시판 시스템을 구축한다.

2. 기능 요구사항

2.1 게시글 (Post)
- 게시글 목록 조회 (페이징 지원)
- 게시글 작성
- 게시글 상세 보기
- 게시글 수정
- 게시글 삭제

2.2 사용자 (User)
- 회원 가입
- 로그인 / 로그아웃
- 작성자만 게시글 수정/삭제 가능
- 비회원도 게시글 조회 가능

2.3 댓글 (Comment) - 선택사항
- 게시글에 대한 댓글 등록
- 댓글 조회
- 댓글 삭제 (본인만 가능)

3. 비기능 요구사항
- DB: MySQL 또는 PostgreSQL
- 백엔드: Spring Boot, Java
- 인증: JWT 기반 로그인
